# Projeto Flask CRUD

Pequeno projeto em Flask com CRUD completo usando SQLite.

## Funcionalidades
- Criar usuários
- Listar usuários
- Editar usuários
- Excluir usuários com confirmação

## Tecnologias
- Python
- Flask
- Flask-SQLAlchemy
- SQLite

## Como rodar
bash
pip install -r requirements.txt
python app.py
